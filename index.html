Static vs Dynamic languages

Static typing

Requires explicit declaration or initialization


							/* C code */
							static int num, sum; // explicit declaration
							num = 5; // now use the variables
							sum = 10;
							sum = sum + num;

Often checked compile time

Dynamic typing

Does not require decleration


							/* Python code */
							num = 10 // directly using the variable

Types are checked runtime
